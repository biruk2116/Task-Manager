# 📝 Task Manager App

A simple and responsive **Task Manager application** built with **React** and **Tailwind CSS**.  
This app helps users organize their daily tasks by allowing them to add, complete, and delete tasks.

---

## 🚀 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 🎨 Clean and responsive UI
- ⚡ Built with React components
- 💾 Fast development using Vite

---

## 🛠️ Technologies Used

- React.js
- Tailwind CSS
- Vite
- JavaScript (ES6)
- HTML5
